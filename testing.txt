Unit Testing Description:
I will test the loadFile() function. This function opens a data file and returns true if it works.

Test Objectives:
    Test if the file opens correctly.
    Test if the function returns true or false properly.

Driver Program:
#include <iostream>
#include <fstream>
using namespace std;

bool loadFile();

int main() {
    bool result = loadFile();
    if (result)
        cout << "Test passed";
    else
        cout << "Test failed";
    return 0;
}

bool loadFile() {
    ifstream file("parts_data.txt");
    return file.is_open();
}
